Set Up and Use GitHub Copilot With VS Code

Introduction

GitHub Copilot is a great way to enhance and speed up your coding ability by leveraging Artificial Intelligence (AI) for code completion and suggestions. This page is designed to help guide ABC123 Corp. software developers through the basics of setting up and using GitHub Copilot.

Prerequisites

Before going further, you must ensure that you have the appropriate accounts set up and programs downloaded. Any steps involving the setup of any prerequisites will not be covered in depth in this guide. Please consult the respective documentation for the following resources if you are unfamiliar.

  • Visual Studio Code: Ensure that Visual Studio Code (VS Code) is installed on your computer. If not, you can download VS Code here.

    *Note: This guide focuses on setting up and using GitHub Copilot with VS Code, but GitHub Copilot officially supports other editors like Visual Studio, Neovim, and JetBrains IDEs. There are also many unofficial plugins for integrating GitHub Copilot with other editors, but results when using unofficial plugins may vary.
  • GitHub: A GitHub account is required for GitHub Copilot integration. If you do not have a GitHub account, you can create one for free on GitHub.com. Once you have a GitHub account, you must ensure your account has the GitHub Copilot permissions enabled. You can follow GitHub's official documentation to complete this step.

    *Note: ABC123 Corp. provides you with a subscription to GitHub Copilot through the company business account. You do not need to purchase a subscription yourself. Access to GitHub Copilot will be granted to you by account administrators. If you are having trouble activating GitHub Copilot on your GitHub account, please reach out to your team manager.

Setting Up GitHub Copilot in VS Code:

  1. Launch VS Code
    • If this is your first time launching VS Code, you may be prompted with additional setup steps. Simply follow the prompts before continuing to the next step.
    • Closing out any currently open projects or code files within VS Code is not required to continue.
  2. Access the Extension Icon
    • Locate the sidebar on the left or right-hand side of your screen.
    • Select the square grid-shaped icon called “Extensions” or use the keyboard shortcut Ctrl+Shift+X on Windows/Linux or Shift+CMD+X on Mac.

    • The Extensions view should now be open.
  3. Search for “GitHub Copilot”
    • In the Extensions view search bar, search “GitHub Copilot” and a few results will appear.

  4. Install GitHub Copilot
    • Click the “Install” button next to the official GitHub Copilot extension (signified by a checkmark below the name).
    • (OPTIONAL) You can also click the GitHub Copilot extension to find more information and to install the extension. The “Install” button is at the top of the page, just below the title.

    • (OPTIONAL) Clicking the arrow next to “Install” will allow you to select which version of GitHub Copilot you want to download. Some newer features may only appear in the pre-release version until those features are officially released.

    • Any of these options will install both GitHub Copilot and GitHub Copilot Chat.
  5. Sign In With GitHub (if you have not already)
    • A GitHub sign-in request will pop up in the bottom right corner if you have not yet signed into GitHub on VS Code.

    • From here, follow the on-screen instructions to finish your GitHub Copilot installation.

Using GitHub Copilot:

  1. Create or Open a File in VS Code
    • For the following steps, you can follow along with an existing code file or create a new file for your chosen programming language.
  2. Begin Coding
    • As you type, you will begin to see GitHub Copilot making code suggestions.

    • Accepting Suggestions - Hit “Tab” on your keyboard to accept the GitHub Copilot suggestion, or hold “Ctrl” and tap the “right arrow key” to start accepting portions of the suggestion.
    • Dismissing Suggestions - Hit “Esc” on your keyboard to dismiss the suggestion or just continue typing to ignore the suggestion
    • A complete list of GitHub Copilot shortcuts for all platforms can be found on the GitHub's official documentation

Accessing GitHub Copilot Chat

  1. Open GitHub Copilot Chat
    • Use the keyboard command Ctrl+I on Windows/Linux or CMD+I on Mac to reveal the GitHub Copilot chat window.
    OR
    1. Right-click inside your open code file.
    2. Find and hover over “Copilot” in the menu.
    3. Click on “Start Code Chat.”

  2. Type Your Question
    • Type your question and hit “Enter” on your keyboard for a response.

    OR
    • Type a “/” slash to see a list of the built-in slash commands.

  3. (OPTIONAL) Expanded Chat
    • After receiving a response to your initial question, click “View in Chat” to open the larger chat window containing all your GitHub Copilot questions and answers.

    • You can make the chat window even larger by clicking the three dots icon at the top of the chat window and selecting “Open Session in Editor.”

  4. (OPTIONAL) Clear Expanded Chat
    • You can clear your GitHub Copilot Chat history by going to the top of the expanded chat window and clicking the “Clear” button with the "three lines and an x” icon.

      Warning: This will instantly delete everything in the chat history. No additional prompts will pop up after clicking the “Clear” button!

Dive Deeper

GitHub Copilot is a fascinating tool that can help boost your productivity as a programmer. Always remember to review and understand the generated code, as this will benefit your own learning and prevent a buggy piece of generated code from being implemented into your project.
