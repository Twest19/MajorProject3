Setting Up and Using GitHub Copilot With VS Code

Introduction

GitHub Copilot enhances and speeds up your coding ability by leveraging Artificial Intelligence (AI) for code completion and suggestions. This guide provides ABC123 Corp. software developers with step-by-step instructions on how to set up and get going with GitHub Copilot.

Prerequisites

Before proceeding, make sure to set up the required accounts and download the necessary programs. Detailed setup instructions for these prerequisites are beyond the scope of this guide. For further guidance, refer to the linked documentation provided throughout.

  • Visual Studio Code: Install Visual Studio Code (VS Code) on your computer if you have not already. Download VS Code here.

    *Note: This guide specifically addresses GitHub Copilot installation and use with VS Code. However, GitHub Copilot also officially supports other editors, including Visual Studio, Neovim, and JetBrains IDEs. Several unofficial plugins exist for integrating GitHub Copilot with other editors, but be aware that experiences with these plugins may vary.
  • GitHub: A GitHub account is required for GitHub Copilot integration. If you do not have a GitHub account, create one for free on GitHub.com. Once you have a GitHub account, ensure you enable GitHub Copilot permissions. Follow GitHub's official documentation for detailed steps.

    *Note: ABC123 Corp. provides you with a subscription to GitHub Copilot through the company business account. You do not need to purchase a subscription. Account administrators will grant you access to GitHub Copilot. If you are experiencing issues activating GitHub Copilot on your account, contact your team manager.

Setting Up GitHub Copilot in VS Code

  1. Launch VS Code
    • If you are launching VS Code for the first time, follow the additional setup prompts before proceeding.

      *Note: You do not need to close any open projects or code files in VS Code to continue.
  2. Access the Extension Icon
    • Locate the sidebar on your screen's left or right side.
    • Select the square grid-shaped icon called “Extensions” or use the keyboard shortcut Ctrl+Shift+X on Windows/Linux or Shift+CMD+X on Mac.

    • This action opens the Extensions view.
  3. Search for “GitHub Copilot”
    • Type “GitHub Copilot” into the search bar in the Extensions view. You will see a few results.

  4. Install GitHub Copilot
    • Click the “Install” button next to the official GitHub Copilot extension (signified by a checkmark below the name).
    • (OPTIONAL) Click the GitHub Copilot extension for more information and the option to install. The “Install” button is at the top of the page, just below the title

    • (OPTIONAL) Click the "down arrow" next to “Install” to choose which version of GitHub Copilot to download.

      *Note: Some newer features might only appear in the pre-release version until officially released.

    • Any of these options will install both GitHub Copilot and GitHub Copilot Chat.
  5. Sign In With GitHub
    • If you have not signed into GitHub on VS Code, a GitHub "Sign-in" request appears in the bottom right corner.

    • Follow the on-screen instructions to finish your GitHub Copilot installation.

Using GitHub Copilot

  1. Open or Create a File in VS Code
    • Open an existing code file or create a new one for your chosen programming language.
  2. Begin Coding
    • As you type, GitHub Copilot makes code suggestions.

    • Accept Suggestions - Press “Tab” on your keyboard to accept the GitHub Copilot suggestion, or hold “Ctrl” and tap the “right arrow key” to accept parts of the suggestion.
    • Dismiss Suggestions - Press “Esc” on your keyboard to dismiss the suggestion or continue typing to ignore the suggestion.
    • *Note: Consult the official GitHub Copilot documentation for a complete list of shortcuts across all platforms.

GitHub Copilot Chat Features

  1. Open GitHub Copilot Chat
    • Use the keyboard command Ctrl+I on Windows/Linux or CMD+I on Mac to reveal the GitHub Copilot chat window.
    OR
    • Right-click inside your open code file.
    • Find and hover over “Copilot” in the menu.
    • Click on “Start Code Chat.”

  2. Ask Your Question
    • Type your question and press “Enter” on your keyboard for a response.

    OR
    • Type a “/” slash to see a list of the built-in slash commands.

  3. Expanded Chat (OPTIONAL)
    • Click “View in Chat” after getting a response to see the larger chat window with all your GitHub Copilot questions and answers.

    • For a larger view, click the three dots icon at the top of the chat window and select “Open Session in Editor.”

  4. Clear Expanded Chat (OPTIONAL)
    • Clear your GitHub Copilot Chat history by clicking the “Clear” button at the top of the expanded chat window, marked by "three lines and an x” icon.

      Warning: This action will instantly delete your entire chat history without further prompts!

Dive Deeper

GitHub Copilot can greatly boost your productivity as a programmer. Remember always to review and understand the code GitHub Copilot generates. This practice enhances your learning and helps avoid implementing buggy code into your project.
